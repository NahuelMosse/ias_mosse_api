# Requests para REST Client

Estos archivos usan la extension REST Client (humao.rest-client).

## Como usarlos

1. Levanta Flask:
   - flask --app app.py --debug run
2. Abre `health.http`.
3. Hace click en `Send Request` arriba de cada bloque.

## Nota

- Endpoint actual: GET /health
- URL base local: http://127.0.0.1:5000
